This research paper explores the relationship between issue salience of welfare and benefit rhetoric on outcomes of the Social Security and Child Support Tribunal in the UK, to test the effcts of media on court proeedings. This repository provides all data materials used to replicate such research and the subsequent code I computed on R Studio to create regressions, which are also deposited for viewing. This includes spreadhseets from government proceedings, which are labelled under SSCS outcomes, on opening this file the sheet used for my research is named under SSCS_2. Additionally, there is another execel sheet comprising my own collected and coded data on salience for the times series frames, and the subsequent coded categories, but also the data from these government tables.
